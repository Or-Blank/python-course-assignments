# Dilution Calculator #
## The calculator: ##
* This calculator is a simplified version based on last week's assignment that help users determine the right concentrations or volumes needed for preparing working buffers, acids, and chemical solutions from concentrated stock.
* This calculator is based on the equation: **C1xV1 = C1xV2**.  
* This calculation is really useful when need to know how much stock solution (V1) to take to prepare a specific volume (V2) of a weaker solution (C2). It can also be helpful when the stock solution (C1) is known and you want to determine how much to dilute it to reach a desired final concentration.
  
## Files: ##
dilution_lib.py - contains the core calculations.

**Dilutions_calculator_input.py** - the code that uses standard input with input(). and imports the calculations from the library.

**Dilutions_calculator_cli.py** - the code that uses command line arguments with sys.argv.

**Dilutions_calculator_gui.py** - the code that uses gui version with tkinter.

**test_dilution_lib.py** - code that test the calculations if they are working.
