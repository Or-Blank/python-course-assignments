## Dilution Calculator ##
# The calculator #:
* This calculator is a simplified version of other calculators whom help finding the right concentrations or volumes of solutions for preparing working buffers, acids, and chemical solutions from concentrated stock.
* This calculator is based on the equation: ~C1*V1 = C2*V2~.  
* This calculation is realy useful when need to know how much stock solution (V1) to take to make a specific volume (V2) of a weaker solution (C2). Or if the stock solution (C1) is known; to calculate how much to dilute it for a certain new concentration.
